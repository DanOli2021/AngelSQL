Política de Privacidad para MyBusiness POS Authorizer

Última actualización: 12 de julio de 2023

Tu nombre de empresa ("nosotros", "nos" o "nuestro") opera la aplicación móvil MyBusiness POS Authorizer (el "Servicio").

Esta página te informa sobre nuestras políticas en relación con la recopilación, uso y divulgación de datos personales cuando utilizas nuestro Servicio y las opciones que tienes asociadas a esos datos.

Recopilación y Uso de Información

Recopilamos varios tipos diferentes de información para varios propósitos con el fin de ofrecerte y mejorar nuestro Servicio.

Tipos de Datos Recopilados

  • Datos Personales: Mientras utilizas nuestro Servicio, podemos pedirte que nos proporciones cierta información personal identificable que se puede utilizar para contactarte o identificarte ("Datos Personales"). La información personal identificable puede incluir, pero no está limitada a, tu dirección de correo electrónico, nombre, número de teléfono, dirección postal.
  • Datos de Uso: También podemos recopilar información que tu navegador envía cada vez que visitas nuestro Servicio o cuando accedes al Servicio a través de un dispositivo móvil ("Datos de Uso").

Uso de los Datos

Tu nombre de empresa utiliza los datos recopilados para varios propósitos:

  • Para ofrecer y mantener nuestro Servicio
  • Para notificarte sobre cambios en nuestro Servicio
  • Para permitirte participar en características interactivas de nuestro Servicio cuando elijas hacerlo
  • Para proporcionar atención al cliente
  • Para recopilar análisis o información valiosa para que podamos mejorar nuestro Servicio
  • Para monitorear el uso de nuestro Servicio

Seguridad de los Datos

La seguridad de tus datos es importante para nosotros, pero recuerda que ningún método de transmisión a través de Internet o método de almacenamiento electrónico es 100% seguro. Aunque nos esforzamos por utilizar medios comercialmente aceptables para proteger tus Datos Personales, no podemos garantizar su seguridad absoluta.
